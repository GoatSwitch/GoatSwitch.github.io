Our solution, GoatSwitch AI, leverages the power of AI to translate outdated or complex code from one programming language into a more modern, efficient one.

A great use case example for GoatSwitch is COBOL migration, a language very common in the financial sector. Many large banks are struggling to find talent and are migrating towards more modern languages like Java.

Our concept of “Test-Verified Code Translation” works as follows:

  1. Generation of accurate tests for the existing code using a LLM.
    We ensure these tests are correct by validating them using the existing code.
  2. Translation of the original code into the desired language using a second LLM.
    Here we generate multiple candidate translations.
  3. Selection of a high quality translation.

The unit tests are used to select a translation from the candidates that is semantically equivalent

to  the original code.

Using this process a user can confidently migrate his code from one language to another, knowing that the new code behaves the same way as the original code.

The role of AI within our solution is twofold:

  1. Test Generation: AI is crucial in parsing through and comprehending the structure, dependencies, and function of the given source code. This allows it to create accurate tests for the code.
  2. Translation: AI also powers our translation process. Modern LLMs can understand and translate any programming language, not only translating the code word for word, but utilizing features of the individual languages.
